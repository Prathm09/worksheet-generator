# **App Name**: Worksheet Wizard

## Core Features:

- Input Collection: Collect user inputs for grade level, curriculum, math topic, and desired difficulty level.
- Problem Generation: Leverage generative AI tool to create 10 unique math problems based on the user-defined parameters. Each question will be tailored to the specified difficulty and topic, incorporating relevant curriculum standards.
- Solution Generation: Use generative AI to provide comprehensive, step-by-step solutions for each generated problem, enhancing the educational value for students and teachers alike.
- PDF Export: Provide functionality to download the worksheet and its solutions as a single, well-formatted PDF document.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) for a sense of intelligence, stability, and confidence, important for an educational application.
- Background color: Very light blue (#F0F4FF), almost white, providing a clean and distraction-free environment for focusing on math problems.
- Accent color: Violet (#8F00FF). It differs significantly from the blue primary color and adds a playful contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif font.
- Use clear, simple icons to represent different subjects and functions within the app.
- A clean, organized layout is important to display math questions and solutions in a clear and readable way.
- Subtle transitions and loading animations to improve user experience during worksheet generation and navigation.
